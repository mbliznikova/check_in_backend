# Check-in App - Backend

This repository contains the backend for Check-In, a web and mobile-friendly app for managing student check-ins, class attendance, and payment tracking for small schools and studios.

## Key Responsibilities

- User authentication & role-based authorization
- Check-in tracking and attendance history
- Student, class, and schedule management
- Payment and reporting data management

## Tech Stack

- Python / Django
- Django REST Framework
- SQLite
- JWT Authentication

## Status

This backend is actively developed as part of a personal project.
